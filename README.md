# Мой первый проект на React (My first project on React)
## ToDoList - список дел (задач)
Возможности:
1) Добавление задач
2) Удаление задач
3) Переключение из "не выполнено" в "выполнено" при клике на задачу

Дополнительно отображается текущее время.
